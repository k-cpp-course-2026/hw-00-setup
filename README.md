# Репозиторий для ознакомления с системой сдачи домашек

## Задание
1. Сделайте форк репозитория  
2. Напишите программу выводящую `Hello world!`
3. Сделайте коммит с решением в ваш форк
4. Для сдачи решения создайте PR в основной репозиторий

## Структура репозитория
- `src/main.cpp` - ваше решение
- `tests` - папка с тестами (одним)
- `CMakeLists.txt` - файл конфигурации сборки проекта (не трогать)

## Как запускать код
1. Создайте папку `build` в корне репозитория
2. Выполните
`cmake -B build -S .`
`cmake --build build`
4. После в папке build у вас должен появиться файл `hw-00-setup`
5. Для тестирования из папки `tests` выполните `python3 run_tests.py  ../build/hw-00-setup .` В выводе должна появится информация о пройденных тестах

**Note:** после каждого изменения исходных файлов (main.cpp) необходимо перекомпилировать программу с помощью `cmake --build build`

## Как сдавать
После того как вы решили задачу локально, необходимо сдать решение, создав PR в основной репозиторий
1. Выберите файлы с вашим решением для коммита. Для данной задачи `git add src/main.cpp`
2. Закоммитьте выше решение к себе в репозитрий с помощью `git commit -m "solution"`, затем `git push`
3. Создайте PR в основной репозиторий
